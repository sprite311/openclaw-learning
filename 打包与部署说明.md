# OpenClaw 打包与部署说明

## 打包给其他电脑使用

### 需要复制的内容

将以下整个文件夹复制到目标电脑：

```
openClaw learning\
├── openclaw\              ← OpenClaw 完整项目（含 node_modules）
├── run_openclaw.bat       ← Windows 批处理启动脚本
├── run_openclaw.ps1       ← PowerShell 启动脚本
├── OpenClaw_Windows安装指南.md
└── 打包与部署说明.md
```

> ⚠️ `openclaw\node_modules` 目录较大（约 1-2 GB），打包时建议用 7-Zip 压缩。

### 目标电脑前提条件

- **必须安装 Node.js 22.16+**（推荐 Node 24）
- 下载地址：https://nodejs.org/
- 无需安装 pnpm、npm 全局包等其他依赖

### 使用方法

**方式 1：直接双击**
- 双击 `run_openclaw.bat` 即可打开

**方式 2：命令行使用**

```powershell
# PowerShell
.\run_openclaw.ps1 --version
.\run_openclaw.ps1 doctor
.\run_openclaw.ps1 gateway run

# CMD
run_openclaw.bat --version
run_openclaw.bat doctor
run_openclaw.bat gateway run
```

**方式 3：直接用 Node 运行**

```cmd
node openclaw\openclaw.mjs --version
node openclaw\openclaw.mjs gateway run
```

## 配置远端大模型

在目标电脑上编辑配置文件 `%USERPROFILE%\.openclaw\openclaw.json`（首次运行会自动创建），添加公司内部模型配置。

详细配置方法请参考 `OpenClaw_Windows安装指南.md` 第 9 节。

---

## 精简打包（可选）

如果觉得 `node_modules` 太大，可以：

1. 只复制源码（不含 `node_modules`）
2. 在目标电脑上运行：
   ```powershell
   npm install -g pnpm
   cd openclaw
   pnpm install
   # A2UI 构建（需要在 PowerShell 中运行以下两条）
   pnpm -s exec tsc -p "vendor\a2ui\renderers\lit\tsconfig.json"
   pnpm -s dlx rolldown -c "apps\shared\OpenClawKit\Tools\CanvasA2UI\rolldown.config.mjs"
   # 主构建
   pnpm ui:build
   pnpm build:docker
   ```
